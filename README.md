## Descrição  

Um programa criado com intuito de estudo e conclusão do projeto de extensão de curso de Informática para negócios da Fatec.
O Sistema serve como um suporte para vendas de um brechó.
O projeto conta com um banco de dados fake [Sem uso de SQL ou qualquer tipo de armazenamento de dados sofisticado]
Possui um CRUD para quase todas as entidades, possibilitando um sistema completo no terminal.

## Tecnologias  
- Java
- Aplicação em terminal (CLI)
